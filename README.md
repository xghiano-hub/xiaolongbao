# Cool Flow - Official Website

Welcome to the official repository for Cool Flow, a creative collective from Rosemont Hills Montessori College.

## 🚀 About the Project
This website showcases our acoustic covers, visual storytelling, and digital content. It features a custom video slider and smooth-scroll animations.

## 🛠️ Built With
* HTML5 / CSS3
* JavaScript (ES6)
* [Lenis](https://github.com/darkroomengineering/lenis) - For smooth scrolling
* YouTube API (Embedding)
